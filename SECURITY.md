# Security Policy

## Supported Versions

Only the latest version of the E-Commerce Management System is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, please report it through one of the following methods:

- **Email**: Send an email to [security@example.com](mailto:security@example.com).
- **GitHub**: Use the "Report a vulnerability" button on the GitHub repository's Security tab if available.

We will acknowledge your report within 48 hours and provide a timeline for a fix. We ask that you follow responsible disclosure practices until a fix is released.
